# mockups

Shows what a design would look like on a real thing — a framed poster on a wall,
a phone post, a card lying on a desk. A club can preview before it prints.

The scenes are drawn by us with simple shapes and shadows (no stock photos, no
AI, no internet) — so the same picture always comes out the same way, and it's
all free and licence-clean. Start at `compose.py`.

Plain-English words: see the top-level GLOSSARY.md.
