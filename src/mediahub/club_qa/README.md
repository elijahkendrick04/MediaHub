# club_qa

Lets a committee member ask questions about the club's own results — like
"when did Ella last PB in 100 Free?" — and get an answer grounded only in the
meets MediaHub has processed, with the meets it used cited. The AI may only
read this club's data through three fixed look-up tools; it never browses the
web and never guesses.
