# radial_rings

A radial competition target built as a ripple: several concentric accent rings
emanate outward from a centre bullseye that holds the verified result — the number
itself, dead centre — with the athlete name seated below the rings and the club
lockup anchoring the foot. The rings thin and fade as they travel outward, so the
eye reads a pulse leaving the centre. The structural signature is the multi-ring
concentric ripple with the result value at the bullseye and the name below. That
makes it deliberately distinct from radial_competition_ring, which is a single
thick ring badge with a punched core, clock ticks and an accent-coloured numeral —
here there is no punched disc, the numeral rides the brand ground, and the target
is a stack of many emanating rings rather than one dial. It is fully typographic,
with no photo.

**When the director should pick it:** a single hero result you want to place at the
literal centre of attention — a standout time, a record mark, a milestone number —
where the radial symmetry and outward ripple lend a ceremonial, focusing feel. It
suits triumphant and precise registers and moments that reward a bold, centred,
almost target-locked emphasis on one figure. Avoid it when the card must carry
several results at once, when a strong athlete photo should lead, or when the club's
tone calls for a calmer, more editorial left-anchored composition.
