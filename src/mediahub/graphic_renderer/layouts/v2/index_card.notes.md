# index_card

A precise **filing card**: a labelled tab up top, a ruled header bearing the name,
then a left-aligned monospace ledger — Event, Result and Detail set as index lines
with dotted leaders running to right-aligned values. The clerical, ruled,
left-aligned register is the structural signature: cooler and more documentary
than the editorial numbers grid's boxed cells, and unlike any centred or photo
layout. Entirely typographic, so it never needs an image.

**When the director should pick it:** record-keeping moments where the facts want
to read as a tidy, official entry — a qualifying time logged, an exact split, a
ratified result, a "for the record" card — and the brand leans clean and
documentary. Reads with no photo and suits "precise", "stoic" and data-led
registers. Avoid for celebratory hype (use a photo-led or big-numeral archetype)
or when there is really only a single number to show.
