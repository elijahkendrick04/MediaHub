# compliance

The data-protection rights engine. Swimmers (and their parents) can say
"don't post about me", "show me what you hold", "fix it", or "delete me" —
this package is the code that actually honours those requests: the consent
registry and its gate (one function asked before any card is approved,
packed, or published), the export/erase/rectify tooling, the retention
purge, the complaints intake, the incident register, and the
pseudonymised security event log.

The paperwork that explains the law behind it lives in
[`docs/compliance/`](../../../docs/compliance/README.md).
