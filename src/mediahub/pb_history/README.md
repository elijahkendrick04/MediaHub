# pb_history

Remembers every swimmer's real past times (from the meets a club uploads) so we
can tell "is this a personal best?" by looking at their **own earlier results** —
instantly, for free, with no website lookup. The more meets a club uploads, the
more it knows.

It never guesses a PB from an "entry/seed time" (those can be wrong). Every time
it stores is a real swum result, kept per club so one club's history never mixes
with another's.
