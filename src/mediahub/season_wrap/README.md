# season_wrap

Adds up a club's stored meet results over a stretch of time (a month or a
season) and turns the totals — PBs, medals, club records, debuts, milestones,
busiest swimmer, biggest improver — into a ready-to-review "month in numbers"
draft pack. Everything here is deterministic counting over saved run files;
captions are written later, at the review surface (W.8).
