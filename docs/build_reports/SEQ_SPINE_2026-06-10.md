## SEQ-2 · Tier B — BUILT (director existed; pool/rank/compliance + route added)

**Found already shipped:** `ai_director.ai_design_spec` (single validated
DesignSpec via `design_spec.normalise`), wired into `generate()`.

**Built this session:**
- `ai_director.ai_design_specs()` — ONE batch call → N mutually-distinct
  validated specs (client-side archetype dedupe; `normalise` is the
  authority).
- `generator`: additive `CreativeBrief.hero_stat_options` +
  `colour_role_assignment`; `apply_design_spec()` — the single spec→brief
  mapping shared by `generate()` and the pool.
- `render.resolved_role_vars_for_brief()` — one resolver: Tier A baseline →
  the director's colour-role assignment applied **only when the whole
  reassigned set clears the APCA gate** (atomic fallback) → medal tint.
- `integration.create_candidate_pool_for_item()` — N specs (batch director,
  deterministic Tier A walk as the honest floor), pool render (one cheap
  format per candidate), per-candidate deterministic compliance verdict
  (APCA pairs + SEQ-0 logo-lockup fit + sponsor-slot presence),
  legibility-first ranking, §8C pool metrics.
- `web.py` create-graphic route: additive `?candidates=N` (2..5) →
  `{candidates, pool_metrics}` with legacy single-visual fields from the top
  candidate; omitting the param keeps the classic response byte-for-byte.

**Verification (live, no mocks):** real Chromium pool n=5 → **5 distinct
archetypes, every candidate compliance score 1.0 (min APCA Lc 101),
archetype_diversity 1.0, perceptual_spread 0.40, 7.9 s wall.** The gate also
caught a real defect during verification: pool briefs initially built at the
card's auto-seed engaged the v1 palette permutation, which can rotate the
brand primary into the accent slot (accent == ground → Lc 0, score 0.0);
fixed by building candidates at seed 0 so direction, not permutation, is the
only axis. `tests/test_gen_v2_tier_b.py` (14 tests): mocked-LLM batch
emission + hallucination normalisation + garbage→None; no-provider floor
(5 ranked deterministic candidates, all scored); gate honour/atomic-reject;
route additive shape; classic-shape regression; variants-worker migration.
