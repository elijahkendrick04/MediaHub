# docs/commercial — sell-side documents (Phase C)

In plain words: paperwork the founder uses to sell MediaHub and to apply for
official data access. Right now it holds the draft application to Swim
England's approved-systems API (the official way to read verified swim times
instead of scraping). The matching in-app tooling is the operator-only
`/operator/commercial` page, backed by `src/mediahub/commercial/`.
